For Part IV, I wrote a Python script using the requests library.

Latency Measurement:
I sent 15 POST requests to /register and 15 GET requests to /list for each instance (US and EU).
For each request, I measured the response time using response.elapsed.total_seconds() and calculated the average latency in milliseconds.


Consistency Test:
I ran 1000 iterations. In each iteration:

Generated a unique username using uuid.uuid4().
Sent a POST request to /register on the US instance.
Immediately sent a GET request to /list on the EU instance.
Checked whether the new username appeared in the list.

I counted how many times the username was missing to measure replication behavior.
The script(src/lm.py) was executed locally and results are reported in Analysis.txt.
